Certainly! Here’s a list of basic SQL operations that you’ll often use when working with databases:

  1. SELECT: Retrieve data from one or more tables.

    sql
    SELECT column1, column2 FROM table_name;

  2. INSERT: Add new rows to a table.

    sql
    INSERT INTO table_name (column1, column2) VALUES (value1, value2);

  3. UPDATE: Modify existing rows in a table.

    sql
    UPDATE table_name SET column1 = value1, column2 = value2 WHERE condition;

  4. DELETE: Remove rows from a table.

    sql
    DELETE FROM table_name WHERE condition;

  5. CREATE TABLE: Define a new table and its columns.

    sql
    CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    ...
);

  6. ALTER TABLE: Change the structure of an existing table (e.g., add or modify columns).

    sql
    ALTER TABLE table_name ADD column_name datatype;
ALTER TABLE table_name DROP COLUMN column_name;
ALTER TABLE table_name MODIFY COLUMN column_name datatype;

  7. DROP TABLE: Delete an entire table and its data.

    sql
    DROP TABLE table_name;

  8. CREATE INDEX: Improve query performance by creating an index on a column.

    sql
    CREATE INDEX index_name ON table_name (column_name);

  9. DROP INDEX: Remove an index from a table.

    sql
    DROP INDEX index_name;

  10. JOIN: Combine rows from two or more tables based on a related column.

    sql
    SELECT columns FROM table1
JOIN table2 ON table1.common_column = table2.common_column;

  11. GROUP BY: Group rows that have the same values in specified columns into aggregated data.

    sql
    SELECT column1, COUNT(*)
FROM table_name
GROUP BY column1;

  12. ORDER BY: Sort the result set by one or more columns.

    sql
    SELECT column1, column2
FROM table_name
ORDER BY column1 ASC|DESC;

  13. HAVING: Filter groups of rows based on a specified condition (used with GROUP BY).

    sql
    SELECT column1, COUNT(*)
FROM table_name
GROUP BY column1
HAVING COUNT(*) > value;

  14. WHERE: Filter records based on specific conditions.

    sql
    SELECT column1, column2
FROM table_name
WHERE condition;

These operations form the foundation of SQL and are essential for interacting with relational databases.
